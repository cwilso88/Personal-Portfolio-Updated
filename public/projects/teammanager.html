TEAM MANAGEMENT APPLICATION

This RESTful API web application is built with React JS, Redux and deployed with Firebase. I created the data using json and utilized the GET, POST, PATCH, and DELETE http methods to complete the actions for the resources within the application. One can view the list of team members, edit the information for each team member, delete the member fom the database, and conditionally add a new member which will be added to the list. The department goals are also information provided by the RESTful API.
